ESP32-S3 — browser flash via Web Serial

1

Enter bootloader mode

  1. Press and hold the BOOT button on the board
  2. Press and release RESET
  3. Release BOOT

No RESET button? Hold BOOT, plug in USB, then release BOOT.

2

Choose firmware variant

Interface selected at power-on by GPIO6 (S0), GPIO7 (S1), GPIO8 (S2) — pull low to activate (internal pull-ups; leave open for HIGH):

S0 (GPIO6)S1 (GPIO7)S2 (GPIO8)Mode
openopenopenComposite — all interfaces active
LOWopenopenKeyboard only
openLOWopenGamepad only
openopenLOWSerial CDC only
LOWLOWopenMIDI only
any other combinationComposite (fallback)
3

Flash

Requires Chrome or Edge on desktop — Web Serial is not available in Firefox. Erases flash and writes firmware (≈30 s).

Web Serial is not supported in this browser. Please open this page in Chrome or Edge.

Raspberry Pi Pico / RP2040 — UF2 drag-and-drop

1

Enter bootloader mode

  1. Hold the BOOTSEL button on the Pico
  2. Plug in the USB cable (keep holding)
  3. Release BOOTSEL — a drive named RPI-RP2 appears
2

Download & copy firmware

Drag the UF2 file onto the RPI-RP2 drive. The Pico reboots automatically.
